📋 경기 요강

⚽ 축구 기본 정보

장소: 운동장 | 인원: 세트당 8명(엔트리 11명) | 경기시간: 20분(추가시간 없음)
1set: 1학년 남학생 / 2set: 2학년 남학생 / 3set: 1·2학년 혼합 남학생

승부 결정

  • 세트 승리 시 청백전 점수 10점 부여
  • 무승부 시 페널티킥 3명으로 승부 결정

세부 규칙

  • 오프사이드 적용하지 않음
  • 전후반은 경기장 중앙 킥오프로 시작, 킥오프 시 한 명 제외 모든 선수 중앙 원 진입 불가
  • 골 후 실점팀 킥오프로 재개
  • 스로인: 양 발 모두 땅에 붙인 상태에서 머리 위 두 손으로 던질 것
  • 골키퍼는 같은 팀 패스를 손으로 잡을 수 없음
  • 라인 아웃: 부심 판단 / 기타 파울: 주심 재량 판단
  • 퇴장 선수는 다음 경기도 참가 불가
  • 두 경기 연속 경고 시 다음 경기 참가 불가
🏐 피구 기본 정보

장소: 농구장 | 인원: 세트당 각 팀 15명 | 3판 2선승제 | 경기시간: 각 게임 7분
1set: 1학년 여학생 / 2set: 2학년 여학생 / 3set: 1·2학년 혼합 여학생
구분을 위해 조끼 착용

경기 방식

  • 내야 12명, 외야 3명으로 시작 / 외야 수비수는 경기 끝까지 외야에만 있음
  • 양 팀 정렬·인사 후 주심 호각 신호와 함께 점프볼로 시작
  • 세트 승리 시 팀 점수 10점 부여
  • 시간 내 미결 시: 내야선수가 많은 팀 승리 (동수 시 최초 1명 탈락까지 진행)
  • 마지막 1분: 공이 밖으로 나갔을 시 시간 정지 후 재개 시 재가동

파울

  • 라인을 밟거나 넘어 던지는 경우 (공격 무효, 공격권 상대팀으로)
  • 헤드샷: 머리·얼굴을 맞힌 경우 (단, 허리 아래 공격 피하려다 맞은 경우 제외)
  • 포 패스: 3회 패스 후 4회째 공격하지 않은 경우
  • 내야-내야 간 패스 불가

기타 규칙

  • 공에 스치기만 해도 아웃 (옷·머리카락 포함, 머리카락 묶기 권장)
  • 아웃된 선수는 공격라인 벗어날 때까지 공 터치 금지
  • 수비 선수는 공격라인 안쪽 공 터치 불가
  • 선수 교체는 사전 제출 명단 내에서만 가능 (부상 제외)
  • 외야 라인 밖 공: 가까운 내야팀에 공격권
🏸 배드민턴 기본 정보

장소: 체육관 | 인원: 각 팀 학년별 남6명·여6명(총 12명) | 혼합 복식 단체전 5점 릴레이
1set: 1학년 / 2set: 2학년 / 3set: 1·2학년 혼합 (1·2set 동시 진행)

5점 릴레이 방식

  • 선수명단 오더지 작성 후 5점 릴레이 혼성 복식 진행
  • 5점 선취 시 승리팀이 2·3번 선수로 교체, 상대팀은 현 1·2번 유지
  • 5의 배수 득점마다 선수 교체 (총 60점으로 종료, 듀스 없음)
  • 첫 경기 서브권: 토스 / 2번째 경기부터 승리팀이 서브권 보유

승부 결정

  • 학년별 경기 승리 시 청백전 점수 10점 부여
  • 기존 배드민턴 규칙 적용 (단, 듀스 없이 진행)
🪢 학급 8자 줄넘기

인원: 18명(남 6명, 여 12명) — 러너 16명, 터너 2명
학년별로 진행, 학급 점수 합산 후 청백팀 집계

  • 줄 끊어지거나 부상자 발생 시 경기 중단, 재측정

점수 배점

참가: 청백팀 전교생 및 교사 전원

💪 줄다리기

인원: 학급당 남학생 3명, 여학생 5명 선발 (총 48명 청백전)
1set: 1학년 48:48 / 2set: 2학년 48:48

경기 방법

  • 경기시간 30초, 3판 2선승제
  • 앉은 채로 줄을 잡고 대기, 신호와 함께 일어나 당기기
  • 30초 전 중앙선과 팀 구분선 모두 넘기면 즉시 게임 종료
  • set당 팀 점수 10점 부여
